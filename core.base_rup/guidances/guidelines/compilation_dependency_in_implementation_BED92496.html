说明

依赖关系的一个重要用途是代表编译依赖关系。编译依赖关系存在于一个元素和编译它所需要的其他元素之间。 例如，在 C++ 中，编译依赖关系用 #include 语句表示。在 Ada 中，编译依赖关系用 with 子句表示。在 Java 中，编译依赖关系用 import 语句表示。通常，不应出现循环的编译依赖关系。
