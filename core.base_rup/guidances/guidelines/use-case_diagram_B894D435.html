说明

具有参与者、用例和它们之间关系的图称为用例图，这些图说明用例模型中的关系。

用例图可组织为用例包（归其所有），且只显示特定包中的相关内容。

使用

关于用例图中应该显示什么并无严格的规定。可以将您认为模型中值得注意的关系显示在图中。以下这些图可能值得考虑：

  • 属于同一个用例包的参与者。
  • 参与者和与该参与者交互的所有用例。这种图可充当参与者的本地图，并很可能与该参与者相关。
  • 处理相同信息的用例。
  • 由同一组参与者使用的用例。
  • 通常在一个序列中执行的用例。
  • 属于同一用例包的用例。
  • 最重要的用例。这种类型的图可以充当模型摘要，并且很可能包括在用例视图中。
  • 一起开发的用例（在相同的增量中）。
  • 特定用例及它与参与者和其他用例的关系。这种图可充当用例的本地图，并很可能与该用例相关。

建议至少在其中一个图中包括每个参与者、用例和关系。 如果使用多个图可以更加清楚地对用例模型进行说明，则可以使用多个图，而且这些图还可以在同一个图中显示多次。
