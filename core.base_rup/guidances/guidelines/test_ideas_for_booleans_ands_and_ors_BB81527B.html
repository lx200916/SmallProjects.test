主要描述

此目录适用于组合了“与”和“或”的表达式。作为一种提示，下面显示了如何得出同类表达式的测试需求：

A1 && A2 && ... && An：

  • 所有 Ai 都为 true 的一个测试构想
  • N 种情况，每种情况都有一项为 false。

示例：

A && B && C
A B C
true true true
FALSE true true
true FALSE true
true true FALSE


A1 || A2 || ... || An：

  • 所有 Ai 都为 false 的一个测试构想
  • N 种情况，每种情况都有一项为 true。

示例：

A || B || C
A B C
FALSE FALSE FALSE
FALSE true FALSE
true FALSE FALSE
FALSE FALSE true
