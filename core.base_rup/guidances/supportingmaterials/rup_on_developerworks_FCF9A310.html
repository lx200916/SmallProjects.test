developerWorks 上的 RUP
developerWorks：Rational 是使用或评估 IBM Rational 工具和最佳实践的开发专业团队的技术资源。
主要描述

developerWorks：Rational 是使用或评估 IBM Rational 工具和最佳实践的开发专业团队的技术资源。developerWorks：Rational 提供各种下载、资源、讨论组和教育，旨在帮助您最大程度地利用 IBM Rational 解决方案。无论您是 IBM Rational 解决方案的初学者还是老手，您都会找到快速入门和提高技能所需的技术内容。

developerWorks：Rational 是更大的 IBM developerWorks® Web 站点的一部分，该站点包括关于 Rational、WebSphere、DB2、Tivoli 和 Lotus 的技术信息以及包括 Java、Linux、XML、Web service、Wireless 以及新兴技术等在内的开放标准。

访问 developerWorks®：Rational® Web 站点的 RUP 部分，可使用技术文章和案例研究，来帮助您采用并实施 RUP、示例工件、诸如 RUP 论坛之类的社区资源等等。 

此外，developerWorks®：Rational® Web 站点的 RUP 部分还是访问 RUP 插件的存储库。 还提供关于使用和创建插件的指导信息。插件扩展了 RUP，并添加了特定于技术、工具、平台或域的流程指导信息，为流程定制提供更多支持。
