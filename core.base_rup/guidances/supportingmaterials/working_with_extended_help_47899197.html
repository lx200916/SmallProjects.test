使用扩展帮助
扩展帮助使您能够查看 IBM Rational Unified Process(R) 指示信息中与您所使用的 Rational 工具相关的主题。
主要描述

概述

扩展帮助使您可以查看关于与您使用的 Rational 工具相关主题的 IBM Rational Unified Process 指示信息。例如，您可以从 Rational Rose 启动扩展帮助而不会中断您的工作流程。

要使用扩展帮助，您必须安装 RUP(R)。安装完成后，扩展帮助就立即可用。

当您从另一个 Rational 工具的“帮助”菜单中选择“扩展帮助”时，“扩展帮助”主题就会出现在 RUP 内容旁边的一个树型面板中。具体的树型面板是由标题“扩展帮助”和工具名称来确定的。“扩展帮助”树型面板列出了适于您正使用的工具及工具中环境的主题。

本文档的其余部分说明如何启动和浏览扩展帮助。

使用扩展帮助

要使用扩展帮助，请遵循以下步骤。

I在 Rational 工具（例如，Rational Rose）中，单击工具菜单栏上的帮助 > 扩展帮助。

如果您安装或发布了多个 RUP Web 站点，则会出现一个扩展帮助位置选择器对话框，提示您选择您要用以查看扩展帮助主题的位置。对于每个 RUP Web 站点，都有一个扩展帮助，并且每个 Web 站点可能有关于您正在使用的工具的不同主题。

在对话框顶部的列表中，选择您所需要的位置。如果需要，您可以选中一个复选框（将选定位置作为下次的缺省值）使该位置成为下次查看扩展帮助时的缺省位置。另一复选框（不再显示此对话框）允许您不再显示对话框。如果您选择永远不再显示该对话框，则缺省的扩展帮助位置随后将是自动启动的扩展帮助位置。

打开一个 Web 浏览器，显示“扩展帮助”树型面板，该面板列出相应的主题。第一页是概述，提供关于扩展帮助的基本信息。例如，如果我们从 Rational Rose 启动了扩展帮助，该树型面板通过选项卡扩展帮助 - Rose 来确定。

先列出最相关的 15 个主题，然后在更多内容文件夹中列出其余的主题。

要查看某一主题，请浏览至该主题并双击以在树形浏览器中展开。“扩展帮助”树型面板仍在左侧，而主题内容出现在右侧。

要查看更多内容文件夹中的主题，请在树中双击以展开该文件夹。

当您想要从其他扩展帮助位置查看主题时，请单击树型面板中的最后一个节点，该节点称为选择其他扩展帮助位置。扩展帮助位置选择器对话框再次出现，允许您选择不同的位置以启动另一个扩展帮助。打开一个 Web 浏览器，并显示相应的“扩展帮助”面板。
