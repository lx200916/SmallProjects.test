工具向导：使用 Rational SoDA 创建设计模型调查
此工具向导描述如何使用 Rational SoDA 创建设计模型调查报告。
工具：Rational SoDA
主要描述

概述

SoDA 使报告生成过程自动化，因而报告能快速并正确地创建。可使用 Microsoft® Word® 或 Adobe® FrameMaker® 版本的 SoDA 生成设计模型调查报告。 要创建此报告，SoDA 将仅收集 Rational Rose 中的设计模型内定义的元素。仅当 Rose 模型遵守设计模型的结构和命名约定时，这才有效。

相关的 Rational Unified Process 信息：

  •  报告：设计模型调查
  • 工作产品：设计模型  

在运行 Windows 2000、NT 4.0、Windows XP、Solaris 或 HP-UX 时此工具向导可用。

要使用 Rational SoDA 创建设计模型调查报告，请使用与您的产品版本对应的过程：

  • 使用 Rational SoDA/Word 生成设计模型调查报告
  • 使用 Rational SoDA/FrameMaker 生成设计模型调查报告

使用 Rational SoDA/Word 生成设计模型调查报告

  1. 从 Rational Rose 的任何地方，单击报告 > SoDA 报告。
  2. 在 SoDA 中显示了可用报告列表时，请选择 Rational Unified Process 设计模型调查报告。
  3. 请单击确定以生成报告。

使用 Rational SoDA/FrameMaker 生成设计模型调查报告

  1. 从 FrameMaker 按钮栏，单击新建。双击 SoDA，然后双击 RoseDomain 并选择 RUPDesignModelSurvey.fm。
  2. 编辑第一个“连接器”并输入模型文件的名称。
  3. 请单击文件 > 另存为，将模板保存到个人目录或项目目录中。
  4. 请单击 SoDA > 生成文档。
  5. 查看已生成的文档。

下一次您想生成此同一文档时，只要打开该文档然后单击 SoDA > 生成文档即可。
