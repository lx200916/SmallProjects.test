工具向导：使用 Rational ClearCase 提升项目基线
本工具向导描述如何使用 Rational ClearCase 统一变更管理（UCM）来对基线指定提升级别。
工具：Rational ClearCase
主要描述

概述

在 ClearCase 中，UCM 基线通常代表组件的稳定配置。

向基线指定了提升级别，这些级别可用来指示基线代表的工作的质量或完成度。通常，提升级别指示测试级别。UCM 提供一组有序的缺省提升级别，还支持用户定义的提升级别。更改基线的提升级别这一操作称为提升或降低基线级别。

建议使用的基线是已对一般用途核准的基线。建议使用的基线用在重新定基操作中。对流重新定基时，会用目标流的建议使用的基线对其进行重新配置。提升级别的用途之一是指定一组具有相同提升级别的基线作为特定流的建议使用的基线。

此工具向导在运行 Microsoft Windows 时适用。

工具步骤

该过程描述如何更改基线的提升级别，以及如何使具有特定提升级别的基线成为流的建议使用的基线：

  1. 更改基线的提升级别
  2. 根据提升级别，将基线设置成建议使用的流基线

有关更多信息，请参阅手册 IBM Rational ClearCase Guide to Managing Software Projects。

1. 更改基线的提升级别

  1. 在 ClearCase Project Explorer 中，选择一个流。单击文件 > 属性，以显示流的属性页。
  2. 单击基线选项卡。
  3. 在组件框中，选择您想要提升其基线的组件。组件的基线显示在基线框中。选择基线并单击属性，以显示基线的属性页。
  4. 在常规选项卡中，单击“提升级别”列表中的向下箭头，以显示所有可用的提升级别。选择新的提升级别。单击确定。

2. 根据提升级别，将基线设置成建议使用的流基线

  • 单击开始 > 程序 > Rational Software > Rational ClearCase > 项目浏览器。
  • 选择一个流。
  • 单击工具 > 建议的基线，以显示当前建议的流基线的列表。
  • 在种子值列表区域中，从下拉列表中选择提升级别。
  • 单击种子值列表，使最近的基线位于或高于建议的提升级别基线。  
  • 单击确定。
