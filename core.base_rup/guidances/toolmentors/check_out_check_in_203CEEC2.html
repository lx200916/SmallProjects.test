工具向导：使用 Rational ClearCase 检出和检入配置项
本工具向导描述如何创建新版本的配置项（检出）以及如何使得新版本的配置项在配置管理数据库中持久存在（检入）。
工具：Rational ClearCase
关系
相关元素
  • 建立配置管理（CM）环境
主要描述

概述

本工具向导描述通过基本 ClearCase 模型进行检入和检出的用途，它适用于运行 Microsoft Windows 的情况。

工具步骤

要使用 ClearCase 检出和检入配置项：

  1. 检出要更改的文件
  2. 作出更改并执行单元测试以验证更改
  3. 检入更改

1. 检出想要更改的文件

  • 浏览至 ClearCase Explorer。从 Windows 工具栏，单击开始 > 程序 > Rational Software > Rational ClearCase > ClearCase Explorer。
  • 在 ClearCase Explorer 中，右键单击要检出的一项或多项，并从 ClearCase 弹出菜单中选择检出。
  • 在检出对话框中输入注释，并单击确定。

  有关检出文件的更多信息，请参阅 ClearCase 手册 Developing Software 或 ClearCase 联机帮助。

2. 作出更改并执行单元测试以验证更改

测试您对已检出的文件或目录的私有副本作出的更改。一旦工作已检入，则可以被其他团队成员访问。

3. 检入更改

  • 浏览至 ClearCase Explorer。 
  • 在 ClearCase Explorer 中，右键单击要检入的一项或多项，并从 ClearCase 弹出菜单中选择检入。
  • 在检入对话框中输入注释，并单击确定。

  有关检入文件的更多信息，请参阅 ClearCase 手册 Developing Software 或   ClearCase 联机帮助。
