工具向导：使用 Rational ProjectConsole 浏览项目工件
此工具向导描述如何使用样本工作产品浏览 Rational ProjectConsole。
工具：Rational ProjectConsole
主要描述

概述

此工具向导描述如何使用随 Rational ProjectConsole 安装的样本工作产品来浏览 Rational ProjectConsole。ProjectConsole 使项目工作产品生成自动化，从而使您可以快速准确地创建项目工作产品。

这一部分提供与此工具向导相关的附加信息的链接。

ProjectConsole 利用 Java applet 技术为您提供了多种您熟悉的导航方式。如果直接单击树控制节点的文本，将显示与该节点关联的工作产品页面。如果通过单击节点的加号展开树控制节点，那么单击节点文本后会出现的超链接将显示为子节点，然后您可以从此处继续浏览。

此工具向导适用于 Microsoft Internet Explorer（发行版 5.5 或更高版本）或 Netscape（发行版 7.0 或更高版本）。

工具步骤

使用 ProjectConsole 浏览项目工作产品：

  1. 打开浏览器。
  2. 输入 ProjectConsole 站点的 URL。可联系 ProjectConsole 管理员以获得 URL。此时将显示 ProjectConsole 登录屏幕。
  3. 在此登录屏幕中输入有效用户标识和密码。此时将显示 ProjectConsole 工作产品浏览器。
  4. 展开以下节点：ProjectConsole、ClassicsCD.com 项目、销售终端（POS）、功能组、管理、发行版 1.0、敏感信息、度量报告、所有发布的度量和功能组。
  5. 单击功能组下面任意节点的文本，以访问该组的度量。
  6. 通过展开以下节点可找到其他工作产品：ClassicsCD.com 项目、Web 商店、发行版 1.0、RUP 规程、项目管理、敏感信息、所有发布的度量和发行版 1.0。展开任意阶段的节点以访问该阶段的度量。
  7. 单击 Rational ProjectConsole 工具栏上的仪表板图标来浏览 ProjectConsole 度量。

注意：ProjectConsole 显示的报告是使用 Rational ProjectConsole Template Builder 创建的样本模板，发布的图表是在 ProjectConsole Dashboard 中创建的。
