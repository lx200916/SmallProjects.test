工具向导：使用 Rational ClearQuest 和 Rational ClearCase 链接配置管理和变更请求管理
本工具向导描述如何能将 Rational ClearQuest 与 Rational ClearCase 统一变更管理（UCM）项目一起使用。
工具：Rational ClearCase
关系
相关元素
  • 建立配置管理（CM）环境
主要描述

概述

ClearCase UCM 项目可以与 ClearQuest 集成，后者是一个变更请求管理工具。通过使用 ClearQuest，您可以提交变更请求，查看和修改现有变更请求，并创建和运行特定于用户或站点的查询和报告。UCM-ClearQuest 集成为 UCM 项目添加了重要的项目管理和活动管理能力。例如，可以将 ClearQuest 记录与 ClearCase 活动和策略（可以设置来控制何时可在 ClearCase 中交付活动以及何时可在 ClearQuest 中关闭活动）链接起来。

此工具向导在运行 Microsoft Windows 时适用。

  有关更多信息，请参阅手册 IBM Rational ClearCase Guide to Managing Software Projects。

工具步骤

要设置 UCM-ClearQuest 集成：

  1. 使 ClearQuest 模式能与 UCM 一起工作，或使用预定义的支持 UCM 的模式
  2. 创建或升级变更请求数据库，以使用该模式
  3. 使 UCM 项目能与 ClearQuest 一起工作

1. 使 ClearQuest 模式能与 UCM 一起工作，或使用预定义的支持 UCM 的模式

以下步骤告诉您如何使用预定义的支持 UCM 的模式来设置变更请求数据库，以与 UCM 一起工作。预定义的 UCM 模式（名为 UnifiedChangeManagement 和 Enterprise），包括记录类型、字段、表单、状态和使用 UCM 项目所必需的其他定义。

要设置变更请求数据库以与 UCM 一起工作：

  1. 创建与一个预定义的支持 UCM 的模式相关联的用户数据库。
  2. 在 ClearQuest Designer 中，单击数据库 > 新建数据库以启动“新建数据库”向导。
  3. 完成向导中的步骤。第 4 步提示您选择与新数据库相关联的模式。 滚动模式名称列表并选择新的模式。
  4. 单击完成。

  有关如何使 ClearQuest 模式能用于 UCM-ClearQuest 集成的信息，请参阅手册 IBM Rational ClearCase Guide to Managing Software Projects 中的“Setting up a Rational ClearQuest user database for UCM”一章。

2. 创建或升级变更请求数据库，以使用该模式

预定义的 UCM 模式允许您立即使用 UCM-ClearQuest 集成，但您可能更希望设计定制模式来跟踪项目活动和变更请求。您甚至更希望使用不同的预定义的模式。依据项目需要，完成此操作有几种方法。

继续之前，建议您熟悉手册 IBM Rational ClearCase Guide to Managing Software Projects 的“Setting up a Rational ClearQuest user database for UCM”一章中的信息。此部分的 ClearCase 文档集中也提供了此任务过程的详细信息。

   有关此过程的详细信息，请参阅手册 IBM Rational ClearCase Guide to Managing Software Projects 中的“Planning how to use the UCM integration with Rational ClearQuest”。

3. 使 UCM 项目能与 ClearQuest 一起工作

  • 在 ClearCase Project Explorer 的左窗格中，右键单击项目以显示其快捷菜单。单击属性以显示其属性页。
  • 单击 ClearQuest 选项卡，然后选择项目支持 ClearQuest 复选框。在 Windows 系统上，为到 ClearQuest 用户数据库的链接提供了一个名称。选择了与该用户数据库对应的连接之后，“Rational ClearQuest 模式存储库”对话框将随即出现。
  • 从“到此变更请求数据库的链接”列表中选择用户数据库。
  • 首次启用项目时，ClearQuest 会打开其“登录”对话框。输入您的用户名、密码以及要将项目链接到的数据库名称。
  • 单击确定。

   高级功能：您还可以为使用 UCM-ClearQuest 集成的项目设置开发策略。请参阅手册 IBM Rational ClearCase Guide to Managing Software Projects 中的“Policies for the UCM integration with Rational ClearQuest”，获取更多信息。
