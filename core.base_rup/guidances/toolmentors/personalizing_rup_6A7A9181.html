工具向导：使用个人流程视图或“我的 RUP”个性化 RUP Web 站点
此工具向导描述如何个性化 Rational Unified Process Web 站点。
工具：Rational Unified Process
主要描述

概述

个人进程视图或“我的 RUP”是 RUP 浏览器的个性化特性。由组织或项目经理创建的进程视图可提供关于 RUP 的集中的、经过整理的概览，这些进程视图在 RUP 浏览器树控件中将作为选项卡出现。您可以创建新选项卡，它们包含 RUP 进程的以更为个人的方式集中的视图，根据您特定的角色、需要和兴趣进行定制。您甚至可以添加在上下文中提供值的新元素。

在本工具向导中执行以下步骤：

  • 选择流程视图
  • 将流程视图另存为个人流程视图或“我的 RUP”选项卡
  • 从选项卡中除去不需要的元素
  • 在选项卡中添加重要元素
  • 组织元素
  • 隐藏流程视图

选择流程视图

进程视图是项目经理创建的 RUP 进程配置的预建立的、不可编辑的视图。RUP 附带有若干缺省的基于角色的视图，但您的组织或项目经理可能创建了新的视图，这些视图对您运行的项目以及这些项目的样式特别有用。

要选择和复审进程视图，请单击位于 RUP 浏览器左边面板内的树控件中的选项卡。如果 RUP 配置包含基于角色的视图，则合理的起始点是能最好地描述您对项目的职责的角色视图。

将流程视图另存为个人流程视图或“我的 RUP”选项卡

选择了要定制的进程视图之后，单击“个人进程视图”或“我的 RUP”工具栏上的“保存”图标（看起来像带有名称的软盘）。为“个人进程视图”或“我的 RUP”选项卡输入名称，并单击“确定”。新的“个人进程视图”或“我的 RUP”选项卡将出现在树控件中，且将位于其他选项卡的顶部。请注意，它上面的图形是一支铅笔。这表示它是一个个人的、可编辑的 RUP 视图。

如果您以与您相关程度最高的角色视图开始，那么您可以合理地用自己的名称保存“个人进程视图”或“我的 RUP”选项卡。

从选项卡中除去不需要的元素

创建定制的 RUP 视图的最简易方法是保存包含您需要的一切内容的进程视图，然后删除与您不相关的元素。

使用标准的鼠标或键盘控件选择您不需要的一个或多个元素。调用右键单击弹出菜单，并选择“删除”选项。

从视图中删除元素。无法撤消操作，但您能通过以下两种方法取回元素：

  • 可以使用树控件工具栏上的“删除”图标（该图标自然地采用废纸箱的图案），删除整个个人进程视图或“我的 RUP”。
  • 可以添加来自任何进程视图树的元素。请参阅下一步以获取指示信息。

在选项卡中添加重要元素

您可以向现有树添加两类元素。

第一类是在现有进程视图中的 RUP 元素，而您希望这些元素出现在个人进程视图或“我的 RUP”中。例如，您可能正在运行一个小型项目，您在项目中既担当项目经理又担当需求分析人员，且您希望所有这些元素出现在一个视图中。  

单击树控件工具栏上的“从缺省值添加”图标（旁边带有加号（+）的树元素）。这将出现一个自由浮点树控件对话框。选择框允许您选择要使用的进程视图。您可以从流程视图中选择一个或多个元素，并将它们拖放到个人流程视图或“我的 RUP”选项卡中。

第二类是来自 RUP 以外的资源，例如与项目相关的 Web 站点，或者存储在可访问磁盘上的文档。

要添加此类链接，您有两种选择。您可以右键单击，从弹出菜单中选择“插入新的”，也可以单击树控件工具栏上的“添加新节点”图标（带有星号的树元素）。它们的行为略有不同，“插入新的”会将元素作为选定元素的子代添加，而“添加新节点”会将元素添加到树的底部。

这两类都具有用于定义元素的相同对话框。它包含用于命名元素的地方，以及元素位置和图标的指针，当打开或关闭元素时您希望使用它们。您可以通过单击这些选项旁边“...”图标或输入 URL 浏览到上述位置或图标。

组织元素

在确保“个人进程视图”或“我的 RUP”选项卡内的元素正确之后，您可以组织它们以向您提供最大价值。例如，您可以将一直担当的角色置于树控件的顶部，而将偶尔担当的角色置于底部。

要将一个元素或一组元素上移或下移一个位置，可使用普通的鼠标或键盘控件选择它们，调用右键单击弹出菜单，并选择“上移”或“下移”。

您还可以将一个元素或一组元素拖放到另一个元素上，以使前者成为后者的子代。这是一个不错的方法，用来将信息保存在“个人流程视图”或“我的 RUP”选项卡中，同时不会使视图散乱。

隐藏流程视图

您可以选择只使一个进程视图可视，以进一步减少散乱现象。要做到这点，可选择您希望可视的进程视图，然后单击工具栏上的“树集”图标。将只有选定的进程视图可视。
