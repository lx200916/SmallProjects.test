工具向导：搜索 RUP Web 站点
此工具向导描述如何搜索 Rational Unified Process Web 站点。
工具：Rational Unified Process
主要描述

概述

当在项目环境中为特定任务或工具的使用寻找流程指导信息时，通常现有的视图不能使您轻易地找到需要的内容。RUP 搜索功能使用一个复杂的引擎，它基于正文中搜索项的接近程度、是否出现在标题中以及出现的次数，来返回经过排序的结果。这使得您能快速地找到需要的材料。  

在本工具向导中执行以下步骤：

  • 启动搜索
  • 输入搜索项
  • 执行搜索
  • 选择搜索结果
  • 更多信息

启动搜索

通过单击 RUP 内容页上部的灰色栏右侧的“搜索”按钮，启动搜索引擎。将显示搜索对话框。

输入搜索项

在右边出现的 4 个框的任何一个中输入搜索项。你的选项有：

  • 任何词：搜索引擎将寻找包含任何一个搜索项的每一页。
  • 所有词：搜索引擎将寻找包含所有搜索项的每一页。
  • 精确短语：搜索引擎将只寻找严格按输入顺序包含搜索项的每一页。
  • 不含词：它与在以上三个框之一中输入的搜索项结合使用。搜索引擎检查以上三个框之一中输入的搜索项得到的结果集，然后废弃包含此框中的项的结果。

您可以选择搜索 RUP 流程配置，或者搜索 developerWorks®：Rational® Web 站点。缺省值是 RUP。

您还可以在结果空间中显示多于或少于 10 个结果（缺省值是 10 个结果）。

执行搜索

单击对话框右边的“立即搜索”按钮。您的搜索结果将显示在搜索项的框下面的搜索对话框中。将显示您所选择数目的结果（缺省为十条）。 您可以选择下一组结果。

如果您选择了 developerWorks：Rational 作为要搜索的部分，将启动一个单独的浏览器窗口，里面含有 developerWorks：Rational。

选择搜索结果

选择与您最相关的搜索结果。如果您在搜索 RUP，则带有该结果的页面将出现在 RUP 浏览器中。选择 RUP 浏览器以查看该页。

如果您在搜索 developerWorks：Rational，则生成的页面将出现在与搜索结果相同的浏览器中。

更多信息  

更多关于搜索选项的详细信息可从 RUP 的搜索帮助页面获取。
