工具向导：使用 Rational ClearQuest 定义变更和复审通知
此工具向导说明当变更请求状态发生变化时，如何设置 Rational ClearQuest&reg 来通知用户或用户组。
工具：Rational ClearQuest
主要描述

概述

ClearQuest 利用电子邮件的方式有两种：电子邮件提交／修改（用户到数据库）和电子邮件通知（数据库到用户）。

电子邮件提交：ClearQuest 管理员使用 Rational E-Mail Reader 让用户通过电子邮件提交或修改记录。Rational 邮件服务器应与 ClearQuest 数据库在同一台机器上运行，以确保电子邮件提交始终可行。

电子邮件通知：ClearQuest 管理员可以配置 ClearQuest 发送电子邮件，通知用户关于变更请求的信息。管理员使用电子邮件规则记录类型来确定电子邮件通知条件、收件人和内容。另外，管理员或用户可以启用客户端机器接收电子邮件通知。即使是 ClearQuest Web 客户端也可以接收电子邮件通知。

工具步骤

1. 使用 ClearQuest Designer 设置电子邮件提交

请参阅 ClearQuest Designer 联机帮助 > 目录与索引 > Administering ClearQuest E-mail> Enabling E-mail Submission。

2. 使用 ClearQuest Designer 设置电子邮件通知

请参阅 ClearQuest Designer 联机帮助 > 目录与索引 > Administering ClearQuest E-mail> Enabling Automatic E-mail Notification。

3. 在 ClearQuest 客户端接收电子邮件通知

请参阅 ClearQuest 联机帮助 > 目录与索引 > Administering ClearQuest E-mail> Enabling ClearQuest Client E-mail Capabilities。

4. 为 ClearQuest Web 客户端设置电子邮件通知

请参阅 Installing Rational ClearQuest。
