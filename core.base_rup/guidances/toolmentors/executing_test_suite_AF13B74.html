工具向导：使用 Rational TestManager 执行测试套件
此工具向导描述如何使用 Rational TestManager 执行测试套件。
工具：Rational TestManager
关系
相关元素
  • 执行测试套件
主要描述

概述

TestManager 的其中一项功能是管理测试工作，可用于将所有类型的测试脚本（GUI、Manual Test、虚拟用户、命令行、Visual Basic 和 Java）整合为一个由用户执行的步骤。该步骤称为套件，可用于在您的测试团队内作为一个粘合的整体（而非零散步骤）分发、执行和评估测试脚本系列。

执行套件时，需要提供特定于运行时的信息。执行分配的套件项的每个虚拟测试员按照这些指示信息运行。执行套件的结果存储在日志中。执行套件之后，可生成报告来分析存储在日志中的数据，以及以图形和图表的形式显示执行结果。

请参阅工具向导：使用 Rational TestManger 设计自动化测试套件，获取有关创建测试套件的更多信息。

在运行 Microsoft Windows 98/2000/NT 4.0 时此工具向导可用。

工具步骤

要使用 TestManager 执行测试套件，需执行以下步骤：

  1. 指定套件
  2. 定义用户负载
  3. 指定日志文件
  4. 启用或禁用资源监视
  5. 选择是否忽略配置
  6. 指定运行时报告选项

1.   指定套件

  • 要指定套件，请从文件菜单单击运行套件。将显示“运行套件”对话框。
  • 要更改套件，请单击套件信息内的更改...。

2.   定义用户负载

  • 如果您选择运行的套件已经定义了固定的计算机组，并且名称下没有可见字段，请转到步骤 3 指定日志文件。
  • 否则，在用户数框中输入要运行的虚拟测试员的数量。
  • 如果套件同时包含固定的和可伸缩的组，则首先分配固定组。可用的虚拟测试员数量取决于您拥有的许可证的类型。如果您的许可证不支持指定的虚拟测试员数，将显示一个错误消息。

注意：在定义了计算机组的情况下，如果选中运行套件前提示计算机复选框，将显示“计算机”列表，并包含编辑计算机、编辑计算机列表和属性按钮。

3.   指定日志文件

当前构建版本号在构建版本框中显示。“日志文件夹”的名称基于套件，而“日志名称”则基于虚拟测试员数和套件的已运行次数。

要更改日志文件夹或日志名称，请单击日志信息内的更改...。

4.   启用或禁用资源监视

如果计划监视资源，则单击监视资源复选框。修改更新速率框中的时间间隔可更改更新视图的速率。时间间隔越小，更新就越快。

监视功能将在您回放套件时监测计算机资源的使用情况，并在您分析结果时，以图形方式表示对应虚拟测试员响应时间内的这些使用情况数据。

5.   选择是否忽略配置

单击忽略测试用例的配置复选框可让 TestManager 忽略测试用例的系统配置，并在任何可用的计算机上运行这些测试用例。

如果选中此选项，TestManager 有三种运行套件的方式：

  • 如果套件包含一个带有已配置的测试用例的测试用例，并且该父测试用例具有实施（例如，测试脚本或套件），TestManager 将在可用的任何计算机上运行此父测试用例，但不运行任何已配置的测试用例。
  • 如果套件包含一个带有已配置的测试用例的测试用例，但该父测试用例没有实施，TestManager 将不运行父测试用例或任何已配置的测试用例。
  • 如果单独的一个已配置测试用例具有实施，TestManager 将在指定的计算机上运行该测试用例。

6.   指定运行时报告选项

  • 要更改 TestManager 报告套件执行状态的方式，请单击选项...。这样就可以设置以下类型的选项：
    • 监视选项卡控制 TestManager 在套件执行过程中显示信息的方式。
    • 报告选项卡指定 TestManager 在套件完成执行时生成的报告。此选项仅适用于性能测试。
    • 运行选项卡控制套件执行的记录，供以后进行评估。
    • VU 编译选项卡指定带条件的 VU 编译的缺省定义，并指定在套件中包含外部 C 库。

   有关更多信息，请参阅 Rational TestManager 联机帮助中标题为 Options dialog box 的主题。

  • 配置了执行之后，单击确定将执行测试。TestManager 检查套件并编译任何未编译或过时的测试脚本。要在 TestManager 检查套件时停止套件执行，请单击取消。
  • TestManager 检查套件并编译必需的测试脚本之后，将最小化“准备运行”窗口。此时不能取消或关闭该窗口。要暂停执行，请单击监视菜单上的停止。
  • 执行测试之后，可以评估测试结果。

   有关更多信息，请参阅 TestManager 联机帮助中标题为 Executing Tests 的主题。
