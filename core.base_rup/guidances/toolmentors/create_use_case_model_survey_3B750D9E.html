概述

SoDA 使报告生成过程自动化，因而报告能快速并正确地创建。可使用 Microsoft® Word® 或 Adobe® FrameMaker® 版本的 SoDA 生成用例模型调查。 仅当 Rational Rose 模型遵守用例模型的结构和命名约定时，这才有效。

相关的 Rational Unified Process 信息：

  •  报告：用例模型调查
  • 工作产品：用例模型  

运行 Windows 2000、NT 4.0、Windows XP、Solaris 或 HP-UX 时可使用该工具向导。

要使用 SoDA 创建用例模型调查报告，请使用与您的产品版本对应的过程：

  • 使用 Rational SoDA/Word 生成用例模型调查
  • 使用 Rational SoDA/FrameMaker 生成用例模型调查

使用 Rational SoDA/Word 生成用例模型调查

  1. 从 Rational Rose 的任何地方，单击报告 > SoDA 报告。
  2. 在 SoDA 中显示了可用报告列表时，请选择 Rational Unified Process 用例模型调查。
  3. 请单击确定以生成报告。

使用 Rational SoDA/FrameMaker 生成用例模型调查

  1. 从 FrameMaker 按钮栏，单击新建。双击 SoDA，然后双击 RoseDomain 并选择 RUPUseCaseModelSurvey.fm 模板。
  2. 编辑该“连接器”并输入模型文件的名称。
  3. 请单击文件 > 另存为，将模板保存到个人目录或项目目录中。
  4. 请单击 SoDA > 生成文档。
  5. 查看已生成的文档。

下一次您想生成此同一文档时，只要打开该文档然后单击 SoDA > 生成文档即可。
