工具向导：使用 Rational TestFactory 评估执行测试套件的结果
此工具向导描述如何使用 Rational TestFactory 评估执行测试套件的结果。
工具：Rational TestFactory
关系
相关元素
  • 执行测试套件
  • 确定测试结果
主要描述

概述

在 Rational TestFactory 中执行了测试套件之后，可评估其结果。有关在 TestFactory 中使用测试套件运行测试脚本套件的信息，请参阅工具向导：执行测试脚本套件。

在运行 98/2000/NT 4.0 时此工具向导可用。

要使用 Rational TestFactory 评估执行测试脚本套件的结果：

  1. 查看测试套件中测试脚本的运行状态
  2. 分析基于代码的测试覆盖率

1. 查看测试套件中测试脚本的运行状态

当测试套件是在 Rational TestFactory 主窗口的左窗格中选中的对象时，右窗格中的状态选项卡将显示运行结果。对于测试套件中列出的各测试脚本，将有一个状态值说明 TestFactory 是否成功执行了该测试脚本。

如果测试脚本未能成功运行，则可在 Rational TestManager 中分析其运行日志。

   请参考 Rational TestFactory 联机帮助中的以下主题：

  • Test Suite- tab
  • Review the run status of a Test Suite
  • View the log for a script run

2. 分析基于代码的测试覆盖率

如果在运行测试套件之前检测了被测试应用程序（AUT），则可查看测试套件基于代码的测试覆盖率数据。当测试套件是在 Rational TestFactory 主窗口的左窗格中选中的对象时，右窗格中的覆盖率选项卡将显示测试套件中运行的所有测试脚本的基于代码的组合覆盖率值。

Rational TestFactory 将把基于代码的测试覆盖率值计算为测试脚本所覆盖源代码相对于被测试应用程序中所有源代码的百分比。如果有权访问 AUT 的源文件，则可使用 TestFactory 中的覆盖率浏览器查看源文件中的代码，并检查测试脚本覆盖了哪些代码行。

也可以查看测试套件中包含的个别测试脚本的基于代码的覆盖率值。 双击测试套件的状态选项卡上的测试脚本将在左窗格中选中该测试脚本。有关详细信息，请参阅工具向导：评估测试覆盖率。

   请参考 Rational TestFactory 联机帮助中的以下主题：

  • Instrumenting the application-under-test
  • Test Suite- tab
  • Review code coverage results for a Test Suite
