概述 

SoDA 使报告生成过程自动化，因而报告能快速并正确地创建。可使用 Microsoft® Word® 或 Adobe® FrameMaker® 版本的 SoDA 生成参与者报告。

相关的 Rational Unified Process 信息：

  •  报告：参与者
  • 工作产品：参与者

在运行 Windows 2000、NT 4.0、Windows XP、Solaris 或 HP-UX 时此工具向导可用。

要使用 Rational SoDA 创建参与者报告，请使用与您的产品版本对应的过程：

  • 使用 Rational SoDA/Word 生成参与者报告
  • 使用 Rational SoDA/FrameMaker 生成参与者报告

使用 Rational SoDA/Word 生成参与者报告

  1. 在 Rational Rose 的用例图中，选择您想记录的参与者。
  2. 请单击报告 > SoDA 报告。
  3. 在 SoDA 中显示了可用报告列表时，请选择 Rational Unified Process 参与者报告。如果列表中没有包含该报告，请取消并确保在用例图中选择了参与者。
  4. 请单击确定以生成报告。

使用 Rational SoDA/FrameMaker 生成参与者报告

  1. 从 FrameMaker 按钮栏，单击新建。双击 SoDA，然后双击 RoseDomain 并选择 RUPActorReport.fm 模板。
  2. 编辑该“连接器”并输入模型、程序包和参与者的名称。
  3. 请单击文件 > 另存为，将模板保存到个人目录或项目目录中。
  4. 请单击 SoDA > 生成文档。
  5. 查看已生成的文档。
