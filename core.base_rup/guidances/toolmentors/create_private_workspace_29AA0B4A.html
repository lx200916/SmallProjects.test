工具向导：使用 Rational ClearCase 创建开发工作区
此工具向导描述如何使用 Rational ClearCase 为个人工作创建私有工作空间。
工具：Rational ClearCase
主要描述

概述

此工具向导描述如何为个人作品创建专工作空间。“私有工作空间”与其他团队成员的工作空间隔开。它为开发人员提供了一个场所，在不必看到或影响其他开发人员的变更的情况下，对系统作出变更。

在 ClearCase 的 UCM 模型中，专用工作空间称为开发工作空间，允许您在与项目其余部分隔离的情况下进行 UCM 活动。通常，每个项目成员在加入 UCM 项目时创建自己的开发工作空间。也可在进行 UCM 项目期间随时创建专用工作空间。

UCM 开发工作空间由流和相关视图组成。UCM 流确定在由该流配置的任何视图中出现的元素版本。各个流也维护一系列 UCM 基线和活动。 视图为诸如编辑和编译源文件、文档格式编排等任务提供工作空间。

此工具向导在运行 Microsoft Windows 时适用。

工具步骤

以下步骤描述在加入 UCM 项目时如何创建开发工作空间。一般步骤为：

  1. 启动“加入项目向导”
  2. 指定流和视图信息
  3. 将视图连接到流

  有关该向导的概述，请参阅 ClearCase 联机帮助中的主题 Join Project Wizard。该向导的每个步骤的详细帮助信息也在每个屏幕的“帮助”主题中提供。

有关 UCM 视图和流的更多信息，请参阅 ClearCase 联机帮助中的这些主题。

  •  About streams 
  •  About views

1. 启动“加入项目向导”

  1. 从 Windows 任务栏，选择开始 > 程序 > Rational Software > Rational ClearCase > Project Explorer。
  2. 从 ClearCase Project Explorer，单击工具箱选项卡 > UCM > 加入项目。出现“加入项目向导”。
  3. 选择要加入的项目并单击下一步，可前进到向导的下一步。

2. 指定流和视图信息

指定开发工作空间的流名称、视图名称、视图的存储位置和视图类型（快照）。“ClearCase 加入项目向导”提供这些值的缺省信息。

注意：“加入项目向导”还将提示您输入为项目的集成工作空间创建视图所需的信息。

  有关视图存储的更多信息，请参阅 ClearCase 联机帮助中的主题 To choose a location for a snapshot view directory。

3. 将视图连接到流

“加入项目向导”提示您确认所作的选择，并连接开发视图和指定的流。单击确定，确认您的选择并继续操作。

  有关关联视图和流的更多信息，请参阅 ClearCase 联机帮助主题 > To attach a new view to can existing stream。
