概述 

此工具向导支持使用 Rational XDE Developer - Java Platform Edition 执行描述分发。

  • 此工具向导中的步骤与相应任务中的那些步骤相匹配。
  • XDE 联机帮助中的主题链接标有 。
  • 在本工具向导中执行以下步骤：
    • 分析分发需求
    • 定义网络配置
    • 将系统元素分配到节点  

分析分发需求

此步骤没有具体的 Rational XDE (TM) 指导信息。

定义网络配置

优化任务：体系结构分析（工具向导：使用 Rational XDE 执行体系结构分析）中创建的部署图。

参考： Working with Deployment Diagrams

将系统元素分配到节点

注释部署图或添加其他图来显示逻辑部署（映射到节点的类、子系统或控制线程）和／或物理部署（映射到节点的文件或文件组）。步骤如下：

  1. 根据需要添加其他部署图。参考： Adding Diagrams
  2. 向图中添加节点或节点实例。参考： Adding Shapes to Diagrams
  3. 将设计模型元素或实施模型元素拖放到图中。参考： Adding Shapes to Diagrams 如果要部署的元素尚未建模，则根据需要将其添加到相应的模型中。（有关各种元素在何种情况下模型的信息，请参阅 Rational XDE Model Structure Guidelines。）
  4. 添加节点和已部署元素之间的部署关系。参考： Adding Deploy Relationships
