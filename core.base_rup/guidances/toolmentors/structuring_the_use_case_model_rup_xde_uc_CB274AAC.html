识别常见的需求 

通用用例的共享可通过添加现有用例之间的关系或者创建新的用例来实现。要实现这一任务，请完成以下步骤：

  1. 打开用例模型。
  2. 浏览到包含用例的包。请参阅 Rational XDE™ Model Structure Guidelines。
  3. 在工具向导：使用 Rational XDE 查找参与者和用例中，创建了包含所有用例和参与者的用例图。浏览到该图。
  4. 重构用例，并根据需要创建新的用例。在工具向导：使用 Rational XDE 查找参与者和用例中描述了使用 Rational XDE 创建新用例的过程。在工具向导：使用 Rational XDE 详细描述用例中可找到这些新用例的详细信息。
  5. 添加用例之间的关系。以下章节描述如何添加每种关系。
  6. 记录每种关系。请参阅 Documenting Model Elements 。

建立参与者间的泛化关系

  1. 打开用例模型。
  2. 浏览到包含参与者的包。请参阅 Rational XDE 模型构造指南。
  3. 在工具向导：使用 Rational XDE 查找参与者和用例中，创建了用以捕获所有参与者的图。打开该图。
  4. 重构现有参与者，并根据需要创建新的参与者。在工具向导：使用 Rational XDE 查找参与者和用例中说明了创建参与者的过程。
  5. 添加参与者之间的泛化关系。请参阅 Adding Generalization Relationships 。
  6. 有选择性地记录每种泛化关系。请参阅 Documenting Model Elements 。

有关更多信息，请参阅 Working with Use-Case Diagrams 。
