比较与同步

Rational XDE 比较数据模型与现有的 DDL 脚本文件或数据库模式。（请参阅 Data Model Comparison 。）然后，可以在模型与 DDL 或数据库之间同步这些差异。（请参阅 Data Model Synchronization 。）您可以使用数据模型到数据库同步向导启动该比较与同步过程。（请参阅 Comparing and Synchronizing the Data Model 。）

Rational XDE 要求您从数据库级别比较与同步数据模型。您正在比较的 DDL 或数据库模式必须与您的数据模型数据库同名。您还必须比较与该数据模型数据库使用相同数据库管理系统（DBMS）的 DDL 或数据库。

请参考 Rational XDE 联机帮助主题 Data Model Compare and Synchronization 。

比较和同步过程需要连接到数据库。以下 Rational XDE 联机帮助主题提供了有关建立特定 DBMS 的数据库连接的信息：

  • Oracle Database Connections
  • SQL Server Database Connections
  • Sybase Database Connections
  • DB2 Database Connections
