工具向导：使用 Rational ClearCase 创建集成和构建工作区
此工具向导描述如何使用 Rational ClearCase 创建集成和构建工作区。
工具：Rational ClearCase
主要描述

概述

集成工作空间的概念类似于实施者的“私有工作空间”，在该工作空间中个人可以在一个包含的区域中开发和更改代码。软件集成人员在集成工作空间中验证单独开发和测试的组件是否可以构建并作为一个产品一起工作。

此工具向导在运行 Microsoft Windows 时适用。

工具步骤

要使用 ClearCase 创建集成和构建工作空间：

  1. 从 Windows 任务栏，通过单击开始 > 程序 > Rational Software > Rational ClearCase > 创建视图转至视图创建向导。
  2. 使用视图创建向导来创建工作空间（ClearCase 视图）并配置它以选择提升的开发变更。将要求提供以下信息：

    • 视图名称
    • 视图位置
    • 根据站点配置而定的其他信息（“视图创建向导”会带您完成所需步骤）

 有关创建视图的详细信息，请参考可从“视图创建向导”获得的联机帮助。

 关于 ClearCase 视图的一般信息（包括配置信息），可在 ClearCase 手册 Developing Software 中获得。
