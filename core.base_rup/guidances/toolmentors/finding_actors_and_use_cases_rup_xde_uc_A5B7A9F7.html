概述 

此工具向导中的步骤与相应任务中的那些步骤相匹配。Rational XDE (TM) 联机帮助中主题的链接标有。

此工具向导假设已根据 XDE 模型构造指南创建了用例模型。

在本工具向导中执行以下步骤：

  • 查找参与者
  • 查找用例
  • 描述参与者与用例的交互方式
  • 封装用例与参与者
  • 在用例图中表示用例模型
  • 开发用例模型的调查
  • 评估结果

查找参与者

找到参与者以后，请记录您的结果，如下：

  1. 打开“用例模型”。
  2. 浏览到将包含参与者的包。请参阅 Rational XDE 模型构造指南。
  3. 创建一个图来包含参与者，为图的类型选择用例图。参考： Adding Diagrams。
  4. 将参与者添加到图中。请参阅 Adding Shapes to Diagrams 。
  5. 以简短描述对每个参与者作注释。请参阅 Documenting Model Elements .

参考： Working with Use-Case Diagrams。

查找用例

找到用例以后，请记录您的结果，如下：

  1. 浏览到将包含用例的包。请参阅 Rational XDE 模型构造指南。
  2. 创建用例图来包含用例。参考： Adding Diagrams。
  3. 将用例添加到图中。请参阅 Adding Shapes to Diagrams 。
  4. 提供事件的简短描述和概要流程。这可以在模型元素上进行初始注释。 （请参阅记录模型元素 。）也可以使用 RUP 为工作产品：用例提供的模板并将其附加到模型中的用例上。请参阅 Linking External Files to Model Elements 。

描述参与者与用例的交互方式

按照下列这些步骤，描述参与者和用例之间的交互：

  1. 将先前创建的参与者添加到包含用例的用例图中。请参阅 Adding Shapes to Diagrams 。
  2. 添加参与者和用例之间的通信关联。（请参阅 Adding Association Relationships 。）如果关联仅在一个方向可导航，请选择有方向的关联，首先单击使用者形状，然后单击提供端形状。
  3. （可选）描述关联。请参阅 Documenting Model Elements .
  4. （可选）添加角色的多重性。请参阅 Specifying Multiplicity in Association Ends 。
