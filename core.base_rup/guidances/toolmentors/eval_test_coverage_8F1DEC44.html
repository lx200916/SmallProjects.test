工具向导：使用 Rational TestFactory 评估测试覆盖率
此工具向导描述如何使用 Rational TestFactory 为 TestFactory 中自动生成的脚本评估基于 UI 的测试覆盖率和基于代码的测试覆盖率。

主要描述

概述

在 Rational TestFactory 中自动生成测试脚本之后，可查看各测试脚本的测试覆盖率信息。有关使用 Pilot 生成测试脚本的信息，请参阅工具向导：自动生成测试脚本。

在运行 98/2000/NT 4.0 时此工具向导可用。

要使用 Rational TestFactory 评估测试覆盖率：

  • 查看测试覆盖率信息。

查看测试覆盖率信息

当测试脚本是在 Rational TestFactory 主窗口的左窗格中选中的对象时，右窗格中的覆盖率选项卡将包含该脚本的测试覆盖率信息。

Rational TestFactory 将基于 UI 的测试覆盖率值计算为测试脚本执行的应用程序（或 applet）中的唯一控件数相对于 Pilot 可访问的控件数的百分比。

Rational TestFactory 将基于代码的测试覆盖率值计算为测试脚本执行的源代码相对于被测试的应用程序中的所有源代码的百分比。如果有权访问 AUT 的源文件，则可使用 TestFactory 中的覆盖率浏览器查看源文件中的代码，并检查测试脚本覆盖了哪些代码行。
