工具向导：使用 Rational ClearCase 比较基线
本工具向导描述如何使用 Rational ClearCase 组件树形浏览器比较 UCM 组件的两条基线。
工具：Rational ClearCase
主要描述

概述

在 ClearCase 中，UCM 基线通常代表组件的稳定配置。基线为一个或多个组件中可见的每个元素确定多个活动和一个版本。ClearCase 组件树形浏览器是一个 GUI，它显示组件基线的历史记录。 您可以用它来比较两条基线的内容。

此工具向导在运行 Microsoft Windows 时适用。

工具步骤

要比较 UCM 项目中的两条基线：

  1. 启动 ClearCase Explorer
  2. 打开指定组件的组件树形浏览器
  3. 选择基线并选择比较参数

进一步的信息请参阅 ClearCase 手册 Managing Software Projects。

1. 启动 ClearCase Explorer

要找到想要查看其基线历史记录的组件，请执行此步骤：

  • 从 Windows 工具栏，单击开始 > 程序 > Rational Software > Rational ClearCase > ClearCase Explorer。

2. 打开指定组件的组件树形浏览器

  • 从组件的弹出菜单中单击浏览基线，以启动 ClearCase 组件树形浏览器。
  • 单击项目的组件文件夹，以显示组件列表。
  • 右键单击组件以显示弹出菜单。
  • 单击浏览基线以启动组件树形浏览器。

请参阅 ClearCase 联机帮助中标题为 ClearCase Component Tree Browser 的主题。

3. 选择基线并选择比较参数

  • 选择一个显示的基线，作为比较的基础。
  • 从工具菜单中，选择一个比较...菜单项。会提示您提供第二条要用在比较中的基线。
  • 您可以查看在活动方面或在版本方面的差别。单击要选择的相应选项卡。
