工具向导：使用 Rational ProjectConsole 在一个图中显示与特定对象相关的工件
此工具向导描述如何使用 Rational ProjectConsole 从 Rational ProjectConsole 显示的图中浏览特定工件。
工具：Rational ProjectConsole
关系
相关元素
  • 执行复审
主要描述

概述

ProjectConsole 可显示图形及文本。如果显示了工作产品图，可使用该图来立即浏览至为图中所显示某项创建的 Web 页面。 例如，模型图可以图形格式显示若干程序包或用例。 此工具向导演示：通过单击图形中显示的任何项（如程序包或用例），可显示该工作产品的页面。

ProjectConsole 利用 Java applet 技术为您提供了多种您熟悉的导航方式。 如果直接单击树控制节点的文本，将显示与该节点关联的工作产品页面。 如果通过单击节点的加号展开树控制节点，那么单击节点文本后会出现的超链接将显示为子节点，然后您可以从此处继续浏览。

使用 Microsoft Internet Explorer（发行版 5.5 或更高版本）或 Netscape（发行版 7.0 或更高版本）时可使用此工具向导。

工具步骤

要使用 ProjectConsole 显示与图中特定对象相关的工作产品：

  1. 打开浏览器。
  2. 输入 ProjectConsole 站点的 URL。可联系 ProjectConsole 管理员以获得 URL。此时将显示 ProjectConsole 登录屏幕。
  3. 在此登录屏幕中输入有效用户标识和密码。此时将显示 ProjectConsole 工作产品浏览器。
  4. 展开以下节点：ProjectConsole、ClassicsCD.com Projects、Point of Sale (POS)、Functional Teams、Development、Release 1.0、Iteration 1、Design - Current View
  5. 单击 ProjectConsole 的右框架中的用例视图超链接。此时将显示“用例视图”工作产品模板，并以“用例视图”信息对该模板进行完全填充。
  6. 单击“用例”视图中的图形可显示其工作产品页面。例如，单击 Point of Sale 程序包将显示 Point of Sale 程序包的内容。
  7. 单击工作产品页面上的任何元素将显示更多模型信息。例如，单击 User Verification 用例将显示工作产品页面，该页面包含有关 Point of Sale 程序包内的 User Verification 用例的信息。

注：ProjectConsole 显示的报告是使用 Rational ProjectConsoleTemplate Builder 创建的样本模板，发布的图表是在 ProjectConsole 仪表板中创建的。
