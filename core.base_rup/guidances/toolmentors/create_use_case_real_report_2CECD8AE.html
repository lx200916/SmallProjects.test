工具向导：使用 Rational SoDA 创建用例实现报告
此工具向导描述如何使用 Rational SoDA 创建用例实现报告。
工具：Rational SoDA
主要描述

概述

SoDA 使报告生成过程自动化，因而报告能快速并正确地创建。可使用 Microsoft® Word® 或 Adobe® FrameMaker® 版本的 SoDA 生成用例实现报告。

相关的 Rational Unified Process 信息：

  •  报告：用例实现
  • 工作产品：用例实现

在运行 Windows 2000、NT 4.0 或 Windows XP、Solaris 或 HP-UX 时可使用该工具向导。

要使用 SoDA 创建用例实现报告，请使用与您的产品版本对应的过程：

  • 使用 Rational SoDA/Word 生成用例实现报告
  • 使用 Rational SoDA/FrameMaker 生成用例实现报告

使用 Rational SoDA/Word 生成用例实现报告

  1. 在 Rational Rose 的用例图中，选择您想记录的用例。
  2. 请单击报告 > SoDA 报告。
  3. 在 SoDA 中显示了可用报告列表时，请选择 Rational Unified Process 用例实现报告。
    如果列表中不包含该报告，请取消并确保在图中选择了用例。
  4. 请单击确定以生成报告。

使用 Rational SoDA/FrameMaker 生成用例实现报告

  1. 从 FrameMaker 按钮栏，单击新建。双击 SoDA，然后双击 RoseDomain 并选择 RUPUseCaseRealizationReport.fm 报告。
  2. 编辑该“连接器”并输入模型、程序包和用例的名称。
  3. 请单击文件 > 另存为，将模板保存到个人目录或项目目录中。
    可能需要更改模板的名称以反映用例实现的名称；例如，MaintainLoansReport.fm。
  4. 请单击 SoDA > 生成文档。
  5. 查看已生成的文档。

下一次您想生成此同一文档时，可打开该文档，然后单击 SoDA > 生成文档。
