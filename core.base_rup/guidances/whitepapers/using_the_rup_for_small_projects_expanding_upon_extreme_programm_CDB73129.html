摘要

RUP 是一个完整的软件开发流程框架，它附带了几个现成的实例。 从 RUP 派生的流程包括轻量级流程（满足生产周期短的小型项目的需要）和更复杂的流程（满足大型的、可能是分布式的项目团队的多种需要）。所有类型和大小的项目都已成功使用了 RUP。此白皮书描述了如何以轻量级方式对小项目应用 RUP。我们描述如何在整个项目的更广环境中有效应用 eXtreme Programming (XP) 技术。
