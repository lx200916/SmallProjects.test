摘要

本文提供了内容管理概述，特别是与在 Web 上交付内容相关的主题。内容管理表示为收集、处理和交付各种内容而结合使用的一组工具和方法。内容管理的范围很广，也有许多挑战。 

收集和处理用于 Web 站点的内容很复杂，而且要花很多时间，这是许多设计过时的文档管理系统所无法处理的。组织已转向将 Web 用作向其客户交付信息和与客户沟通的方式；但只将内容上载到 Web 站点并不能保证它将转到适当的人手中，也不能保证将满足他们对信息的需求。

与购买或构建提供内容工作流程的应用程序相比，内容管理项目有更高的要求。内容管理解决方案的开发是一组相关的软件项目（由数据库、生产工作流程、内容交付和 Web 应用程序构成），这令它非常适合诸如 RUP 之类已建立的软件开发框架。
