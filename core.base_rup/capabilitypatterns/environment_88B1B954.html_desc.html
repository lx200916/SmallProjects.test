决定如何执行工作流程

应就环境规程的工作流程做出以下决策：

  • 查看此工作流程中的活动，然后决定如何执行工作流程。查看图的警戒条件和指导信息。决定要执行哪些活动并以何顺序执行。 
  • 决定要执行环境活动的哪些部分。通常，环境规程中的工件是在需要时引入的。 例如，只有在项目将制定 用户支持材料的情况下，才会制定 手册风格指南。
  • 决定在项目生命周期内何时引入工作流程的每个部分。开发流程始终是在开始项目时引入。同时，还可以选择使用 开发案例来记录详细的、特定于项目的定制决策。 定制的 交付流程可以导入到 Rational Portfolio Manager 中，用于辅助进行工作分配和项目进度跟踪。 其他工件在需要时引进。
