活动：支持迭代期间环境
此活动的目的是在当前的整个迭代中支持项目的开发环境，开发环境同时包含流程和工具。
描述工作分解结构团队分配工作产品使用
用途
此活动的目的是为开发人员在迭代期间使用工具和流程提供支持。
关系
父代活动
  • 环境
描述

支持项目环境是一项持续进行的任务，它使项目成员可以有效地完成他们的工作，而不会由于开发环境的问题而发生延误。这包括安装必需的软件，确保硬件正常运行，以及确保潜在的网络问题及时得到解决。

人员配备

环境支持由单个人员或若干人员组成的小型团队执行。在大型组织中，这些人员通常是公司 IT 部门的员工或其流程组的成员。

使用
使用指导信息

项目环境需要持续关注。这表示当整个项目机构已启动并在运行时，与支持环境相关的任务在项目的每次迭代过程中执行，并在精化阶段晚期和构造阶段早期达到峰值。

大多数项目设置中都需要此活动。
