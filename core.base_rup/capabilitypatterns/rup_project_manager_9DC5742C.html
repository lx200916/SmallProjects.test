角色：项目经理
该角色规划、管理和分配资源、设定优先级、协调与客户和用户的沟通，并使项目团队的集中于其工作目标。项目经理还建立一套工作方法，以确保项目工作产品的完整性和质量。

主要描述

项目经理将项目管理知识、技能、工具和技术广泛应用于各种任务，以满足需求并交付特定项目的最终结果。

技能

要胜任项目经理角色，建议您具备以下技能：

  • 软件开发生命期、应用程序领域和平台方面的经验
  • 范围估计、规划、时间管理、调度、项目成本估算和预算管理
  • 资源规划、资源管理和采购
  • 风险分析、相关性和决策分析技能
  • 展示、沟通和协商技能
  • 项目管理方面的经验
  • 领导层和团队建设能力
  • 冲突解决、问题解决技能，以及在压力下作出合理决策的能力
  • 基于可交付件的管理，关注客户价值的实现（以执行符合或超出客户需要的软件的形式）。
分配方法

对于较小的项目，一个人可兼任项目经理和开发角色（例如软件设计人员）。 但是，如有可能，通常最好不要让项目经理承担开发职责，以确保管理职责方面的时间压力不会妨碍到开发任务，反之亦然。

项目经理角色通常可以与其他管理型角色成功地组合在一起，例如变更控制管理员、部署管理员和流程工程师。

项目经理可能需要对诸如收集项目状态信息、生成度量标准和准备报告之类的任务的支持。当为项目配备人员时，请考虑将支持人员包括在内，以帮助执行这些任务。
