主要描述

IBM® Rational® SoDA® 应用程序能自动生成软件文档。Rational SoDA 模板支持 Microsoft® Word® 97、2000 和 2002 (XP)。SoDA 模板还支持 Adobe® FrameMaker®+SGML 5.5.6（在 UNIX 和 Windows 上）以及 FrameMaker+SGML 6.0（仅在 UNIX 上）。
