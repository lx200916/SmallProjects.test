主要描述

IBM® Rational Unified Process® 平台是一个灵活的软件开发流程平台。通过可配置的体系结构，RUP 平台使您能够只选择和部署项目的每个阶段所需要的流程组件（包括角色、活动、模板、指南和工具向导）。通过处于其核心位置的经过业界证实的软件工程最佳实践，RUP 平台包含了用于针对项目的特定需要配置 RUP 的工具，将自己的内部知识开发到流程组件中的工具，强大且可定制的基于 Web 的部署工具，以及用于与同行和业界领导者交换最佳实践的网上社区。通过使用经过证实的方法和共享单个综合流程，您的团队将能够更有效地通信，更高效地工作。

RUP 浏览器是进入到定制流程配置的窗口，它允许您查看它的不同视图，创建您自己的视图，搜索内容，以及通过扩展帮助访问其与您在工具中的位置最相关的元素。
