关系
工具向导
  • 使用 Rational PurifyPlus 工具分析运行时性能（Windows 和 UNIX）
  • 使用 Rational PurifyPlus 工具执行测试套件（Windows 和 UNIX）
主要描述

IBM® Rational® PurifyPlustm 应用程序是先进的调试和诊断工具，具有以下功能：

  • 指出应用程序中难以查找的运行时错误。
  • 提供先进的应用程序性能概要分析；
  • 指出运行时执行过程中未操作的代码区域。

尽管 Rational PurifyPlus 通常由开发人员在测试和调试活动期间使用，它也可以整合到由单独的测试员执行的测试活动中。
