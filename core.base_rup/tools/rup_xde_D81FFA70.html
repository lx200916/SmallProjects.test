主要描述

IBM® Rational® XDEtm 是图形化组件建模和开发工具，它使用业界标准的统一建模语言（UML）。

以下工具向导涉及 Rational XDE Developer - Java Platform Edition 和 Rational XDE Developer - Microsoft .NET Edition。两个平台之间有差异的地方都有说明。

概述

Rational XDE Developer - Java Platform Edition 将您的设计与 Java 开发组合到一个紧密集成的环境中。仅一个环境就避免了与环境外的工具进行切换的麻烦。

Rational XDE 改进了您可以使用的许多工作方式。例如，它允许您：

  • 在同一环境中从分析和设计一直转至代码。
  • 指定模型和代码之间的手动和自动同步。
  • 定义代码模板和模型模板，以节省时间并有助于执行标准。
  • 创建 UML 和自由格式模型，并验证模型的 UML 一致性。
  • 在同一项目中构建可以相互跟踪的多个模型。
  • 复用已定义的模式，以在团队中共享代码。

Rational XDE Developer - Java Platform Edition 可以与以下任一 Websphere Studio 平台产品一起使用：

  • IBM Websphere Studio Workbench (WSW)
  • IBM Websphere Studio Application Developer (WSSAD) － 类似于 WSW，但包含适用于 Java (TM) 2 Enterprise Edition (J2EE) 和 Web service 开发的其他功能部件。

这两者都是工具集成和开发环境，由 Rational XDE 环境的透视图、视图、编辑器、菜单和其他构建块组成。通常，您使用多个透视图：

  • Java 透视图包含 Java 编辑器、类查看器和其他几个编辑器。
  • 建模透视图包含模型浏览器、工具箱和其他视图。

Rational XDE 与这两个产品环境所执行的操作几乎完全相同。在工具向导中注明了一些微小的区别。
