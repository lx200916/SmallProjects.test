概念：模板
此指导信息类型指定工作产品的结构，它提供预定义的目录、部分、包和／或标题、标准化格式以及有关如何使用和完成部分／包的描述。
主要描述

模板是工作产品的“模型”或原型。在工作产品描述中，通常有一个或多个模板可用来创建对应的工作产品。模板与用来创建工作产品的工具链接。在项目开始前，组织可能会想要定制模板，即添加公司徽标、一些项目标识或特定于项目类型的信息。

示例：

  • Microsoft® Word® 模板可用于文档类工作产品，也可用于某些报告。
  • 用于 Microsoft Word 或 Adobe® FrameMaker® 的 Rational SoDA 模板会从诸如 Rational Rose、Rational RequisitePro 或 Rational TeamTest 之类的工具中抽取信息。
  • 用于流程的各种元素的 Microsoft® FrontPage® 模板。
  • 用于项目计划的 Microsoft Project 模板。
