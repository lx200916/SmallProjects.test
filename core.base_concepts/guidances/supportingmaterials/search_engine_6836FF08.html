搜索引擎
搜索引擎允许您在发布的 Web 站点中搜索页面。
主要描述

注：作为 applet 实现的搜索引擎需要 JRE 1.4.2 或更高版本（可从以下地址下载 JRE：http://java.sun.com/j2se）。

使用搜索引擎的技巧

搜索引擎允许您使用多种方法来在发布的 Web 站点中搜索页面。例如，您可以：

  • 搜索包含输入的所有词的页面。
  • 搜索包含输入的任意词的页面。
  • 搜索包含输入的精确短语的页面。
  • 搜索不包含输入的所有词的页面。

要输入搜索查询，则根据您的选择（所有词、任意词、精确短语以及不含该词字段）输入要搜索的词，然后按 ENTER 或单击立即搜索按钮。搜索完成后，将在结果字段中列出每个匹配的页面，显示页面标题及内容的简短摘要。单击某个标题以在发布的 Web 站点浏览器窗口中打开页面。

例如，要搜索同时包含“Rational”、“Unified”和“Process”这三个词且包含“adopt”和／或“vision”的页面，则在所有词字段中输入 Rational Unified Process，并在任意词字段中输入 adopt vision。

您可使用显示列表来选择每页显示的结果数目。如果结果的数目超出所选的每页显示数目，则单击下一页和上一页按钮来翻阅结果。

您还可指明是对发布的 Web 站点还是 developerWorks 应用查询。要在发布的 Web 站点和 developerWorks 之间做出选择，可以单击节中列表，然后选择期望的节。

查找页面中的词

一旦搜索引擎显示了某个页面，则使用 Web 浏览器的搜索工具在该页面上查找特定的词。 按 CTRL+F 可启动 Web 浏览器的搜索工具。

输入搜索查询

搜索查询由一个或多个指定的词组成。不能使用布尔运算符。请使用所提供的所有词、任意词、精确短语或不含该词字段，不要使用布尔运算符。搜索过程是不区分大小写的，这意味着 Hello、HELLO 和 hElLo 都被视为同一单词。不支持通配符 *。

当使用多个字段时，将按从上到下的优先顺序对查询求值。例如，查询：

所有词： project management
任意词： adopt vision
精确短语： Rational Unified Process
不含该词： implementation


这可如下表示：(project AND management) AND (adopt OR vision) AND (Rational Unified Process) NOT (implementation)。换句话说，匹配的页面必须同时包含“project”和“management”这两个词、包含“adopt”或“vision”，以及短语“Rational Unified Process”。匹配的页面不能包含词“implementation”。

特别注意事项

  • 由于在创建索引文件（搜索将基于索引文件操作）期间排除了“where”、“when”和“the”之类的常见词，所以搜索引擎会自动将这些词从搜索查询中排除。排除这些词将提高搜索的成效，而不会影响结果的精确性。
  • 为了让使用不含该词字段的查询有意义，必须至少在另外的一个搜索字段中输入文本。换句话说，除非您首先指定希望搜索到确实包含特定词或特定短语的页面，否则搜索引擎将无法找到不包含特定词的页面。
  • 不支持使用通配符进行通配符搜索。
  • 不支持布尔运算符。关于如何执行与使用布尔运算符等效的搜索的指示信息，请参阅输入搜索查询一节。
  • 如果尝试以数字形式搜索单个数值（特别是 0 至 9 的数值），搜索结果可能会不令人满意。应在搜索中省略数字或使用数字的完整文本拼写（例如“zero”、“six”、“nine”、“ten”等），而不是搜索数值。
