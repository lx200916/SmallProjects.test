主要描述

方法 Web 站点包含一组 HTML 页面，您可以使用 Microsoft® Internet Explorer、Mozilla 和 Firefox 浏览器进行查看。

注意：一些作为 applet 实施的功能（如“搜索”和“我的视图”）需要 JRE 1.4.2 或更高版本（可从 http://java.sun.com/j2se 下载 JRE）。 或者，您也可以发布不带 applet 的配置，该配置不具有“搜索”和“我的视图”功能，但也不需要 JRE。如果您要查看的站点没有通过 applet 发布，则将不显示“搜索”和“我的视图”。

选择该图的任意区域以获取如何进行浏览的简要说明。

浏览器环境的元素

词汇表

单击此按钮将启动一个包含词汇表的单独窗口，它以字母顺序列出了在方法 Web 站点中使用的术语以及定义和示例页面的链接。

索引

单击“索引”按钮将启动一个单独窗口，其中包含方法 Web 站点中的一列按字母顺序排列的主题。 单击索引中的超链接将使相关页面在主窗口中显示。

反馈

单击“反馈”按钮将启动一个单独窗口，其中包含要发送到 Rational 反馈的电子邮件消息，该邮件自动引用当前在主窗口中显示的页面。

关于

单击“关于”按钮将启动一个弹出窗口，其中包含版权、当前版本号。

搜索

“搜索”允许您输入关键字并在流程中进行搜索，然后将所有与该主题相关的页面显示在“搜索结果”窗口中。“搜索实用程序”处理关键字主题，而非在流程内容页面中搜索字符串。
请注意作为 applet 实施的“搜索引擎”需要 JRE 1.4.2 或更高版本（可从 http://java.sun.com/j2se 下载 JRE）。 如果您要查看的站点没有通过 applet 发布，则将不显示“搜索”。

打印

“打印”按钮将主窗口中的内容发送到您的打印机。

主要内容框架

此框架显示方法内容。将内容组织在可展开和折叠的部分中。 

视图

视图使您可以从另一个观察角度浏览并使用方法 Web 站点。以下是视图的一些示例。

  1. 团队视图显示在 Web 站点中的所有内容。它包含所有其他视图，并允许您从不同的观察角度（生命周期、交付流程、角色、规程和工作产品等）浏览方法。它还提供“入门”信息。
  2. 入门视图提供对方法基础（主要原则、必需流程和方法体系结构基本原则）的说明、关于如何浏览 Web 站点的信息以及从特定技术或项目干系人的观察角度使用 Web 站点的指南。它还提供附录类的材料，例如参考和资源。
  3. 角色集视图允许您选择感兴趣的角色，并查看与角色关联的所有工作。您还可以查看对此角色的工作很重要的规程、域和工具向导。
  4. 交付流程视图允许您浏览生命周期（诸如经典 RUP 和业务建模生命周期）。树浏览器包含了从阶段级别直到迭代、活动和子活动级别的生命周期元素。为每个生命周期元素提供了三个细分结构：
    • 显示活动和任务的工作分解结构
    • 显示角色的角色分配
    • 显示工作产品的工作产品使用

请注意工作分解结构可能具有可导航的图形，该图形说明关联生命周期元素中包含的工作。还请注意工作分解结构中的任务描述符包含步骤。 这些步骤显示为蓝色实心圈。蓝色空心圈表示在此生命周期中，该步骤不会完成。
